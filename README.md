### 🧭 저는 이런 것을 만듭니다

- **로컬 LLM로 비용 0원 파이프라인** — LM Studio·KURE-v1 임베딩과 n8n을 엮어 채용공고 수집·요약·의미검색을 자동화합니다.
- **RAG 시스템** — LangGraph·ChromaDB·pgvector 기반으로 문서를 검색·응답하는 챗봇을 설계합니다.
- **모델 성능 검증** — GPU에서 추론 처리량을 직접 벤치마킹하며 "느낌"이 아닌 숫자로 판단합니다.

> API를 뜯어보고 다중 백엔드를 자동 탐색하는 걸 좋아합니다. 동작하지 않으면 로그를 파고들어 근본 원인을 찾을 때까지 멈추지 않습니다.
